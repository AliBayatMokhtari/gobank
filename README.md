# Core features

- [ ] Authentication
- [ ] Create wallet for user when user's national code is verified
- [ ] User can charge their wallet
- [ ] User can cash out from their wallet (Deposit to bank account)
- [ ] User can transfer credit to other user wallet (With wallet public id)
- [ ] User should see list of transactions in details
  - Transfer date
  - Destination wallet's owner name
  - Transfer amount
  - Transfer status
- [ ] User can filter their transactions based on date range or transaction status
